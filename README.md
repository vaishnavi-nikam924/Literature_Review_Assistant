# 🤖 Agentic Literature Review Assistant

An intelligent multi-agent system that automates literature reviews using **Agentic AI, Retrieval-Augmented Generation (RAG), and Reinforcement Learning**.

---

## 📌 Overview

With the exponential growth of academic research, manually conducting literature reviews is time-consuming and error-prone. This project introduces an **Agentic Literature Review Assistant** that:

- Retrieves relevant research papers
- Generates structured summaries
- Evaluates output quality
- Iteratively refines results using self-reflection

Unlike traditional NLP pipelines, this system is **adaptive, autonomous, and iterative**.

---

## 🧠 Key Features

✨ **Multi-Agent Architecture**
- Planner Agent (query decomposition)
- Retriever Agent (hybrid search)
- Summarizer Agent (structured summaries)
- Evaluator Agent (quality scoring)
- Orchestrator Agent (workflow control)

🔍 **Hybrid Retrieval System**
- BM25 (keyword-based search)
- FAISS (semantic vector search)
- Weighted score fusion for improved accuracy

🔁 **Iterative Self-Refinement**
- Chain-of-Thought reasoning
- ReAct (Thought → Action → Observation loop)
- Feedback-driven query improvement

📊 **Multi-Dimensional Evaluation**
- Perception
- Reasoning
- Tool Usage
- Coherence
- Grounding (Hallucination detection)

🛡️ **Security Layer**
- Prompt injection detection
- Tool misuse prevention
- Human-in-the-loop validation

🤖 **Reinforcement Learning Optimization**
- PPO (Proximal Policy Optimization)
- DPO (Direct Preference Optimization)
- GRPO (Group Relative Policy Optimization)
